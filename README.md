# 🌿 Trips to Dandeli – Adventure Travel Website

**Trips to Dandeli** is a modern travel website built using **HTML, CSS, and JavaScript** to showcase adventure tourism packages, destinations, and travel experiences in **Dandeli, Karnataka**.

The project includes a **blog system, privacy pages, and dynamic JavaScript logic using the accumulator pattern** to manage blog data and content.

## 📌 Features

* 🌿 Adventure travel landing page for Dandeli
* 📱 Fully responsive design for mobile and desktop
* 📝 Blog section implemented using **JavaScript accumulator pattern**
* 🧭 Travel packages and destination information
* 📞 Contact Us page for inquiries
* 🔒 Privacy Policy page
* 📄 Terms & Conditions page
* 🎨 Smooth UI layout and clean navigation
* ⚡ Lightweight and fast-loading website
